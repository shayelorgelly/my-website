FizzBuzz

What is FizzBuzz?

For people unfamiliar with the concept, FizzBuzz is a coding challenge often used in interviews. The task is to write a program that prints numbers from 1 to 100 while replacing multiples of 3 with "Fizz", multiples of 5 with "Buzz" and multiples of both 3 and 3 with "FizzBuzz". It may sound simple, and it is! However, it is a good way to show a programmers overall coding proficiency and problem-solving abilities.

My Journey with FizzBuzz

It all started one evening when my friend introduced me to the programming challenge known as FizzBuzz. Whether it was from my quest for more coding knowledge or just pure boredom, I immediately felt motivated by the challenge presented. My friend introduced me to the basics and laid out the rules. I thought it would be a easy task however as I refined the code further it got a little more complex than I anticipated. I began experimenting with my favorite programming language, JavaScript and after a few tests, i had written a script which looped through all the numbers, divided them by 3 and 5 and checked if the results were integers or float-point numbers. 

for (let i = 1; i <= 100; i++) {
    if (Number.isInteger(i / 3) && Number.isInteger(i / 5)) {
        console.log("FizzBuzz");
    } else if (Number.isInteger(i / 3)) {
        console.log("Fizz");
    } else if (Number.isInteger(i / 5)) {
        console.log("Buzz");
    } else {
        console.log(i);
    }
}
